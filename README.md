# 🎓 VCET AI Smart FAQ Chatbot

An AI-powered college FAQ chatbot built using *Python, Streamlit, and Ollama (Llama3)*.

The chatbot answers questions related to:
- VCET college information
- Branch details
- Syllabus
- Admissions
- Placements
- Facilities
- Hostel
- Courses like AIML, CSE, CD, MBA, etc.

---

# 🚀 Features

✅ AI-powered chatbot using LLM

# ⚙️ Installation

## 1. Clone Repository

bash
git clone https://github.com/YOUR_USERNAME/vcet-ai-chatbot.git


## 2. Open Project Folder

bash
cd vcet-ai-chatbot


## 3. Install Requirements

bash
pip install -r requirements.txt


## 4. Install Ollama

Download:
https://ollama.com/download

bash
ollama run llama3

# 🧠 How It Works

1. User enters question
2. Streamlit captures input
3. Python creates prompt
4. Prompt sent to Llama3 using Ollama
5. AI generates response
6. Response displayed on dashboard
